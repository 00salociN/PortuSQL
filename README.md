# PortuSQL

PortuSQL é uma IDE desktop educacional para escrita e teste de consultas SQL, desenvolvida em Java com JavaFX. O projeto tem como objetivo oferecer um ambiente leve, moderno e didático para estudantes e iniciantes em banco de dados.

A aplicação simula a experiência de ferramentas profissionais de banco de dados, mantendo foco em simplicidade, organização visual e extensibilidade.
